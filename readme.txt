Prj sử dụng xampp, vscode

Mặc định host trên đường dẫn $_DOMAIN = http://localhost/SoundDreamer/index.php

Tuỳ chỉnh biến $_DOMAIN theo đường dẫn host tại các file sau:
    1. core/init.php
    2. jscript/form.js
    3. jscript/func.js

Các thư mục:
    1. ajax-php : chứa các file nhận yêu cầu ajax
    2. classes : chứa các file class sử dụng trong trang web
